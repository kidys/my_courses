###Изучаем конвертер `markdown` в `html`

__Markdown__ - это облегчённый язык разметки, созданный с целью написания наиболее читаемого и удобного для правки 
текста, но пригодного для преобразования в языки для продвинутых публикаций.

##Пример кода в формате `php`

~~~
$a = 3;
$b = 2;

function sum($a, $b) {
    return (int)$a + (int)$b;
}

function multiply($a, $b) {
    return (int)$a * (int)$b;
}

echo sum($a, $b);
echo multiply($a, $b);
~~~
{:php}
